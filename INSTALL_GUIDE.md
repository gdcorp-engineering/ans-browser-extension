# Simple Installation Guide for Non-Technical Users

This guide is designed for users who are not familiar with technical processes. Follow these steps carefully.

## üì• Step 1: Download the Extension

1. Download the extension ZIP file (it will be named something like `ans-extension-dev-v1.5.4.zip`)
2. The file will usually be saved to your **Downloads** folder
3. Remember where you saved it!

## üìÇ Step 2: Extract the ZIP File

You need to "unzip" or "extract" the file to get the extension folder.

### On Windows:
1. Find the ZIP file in your Downloads folder
2. **Right-click** on the ZIP file
3. Click **"Extract All..."**
4. Choose where to extract (Desktop is a good choice)
5. Click **"Extract"**
6. You'll see a new folder appear

### On Mac:
1. Find the ZIP file in your Downloads folder
2. **Double-click** the ZIP file
3. It will automatically extract to the same location
4. You'll see a new folder appear

### On Linux:
1. Find the ZIP file in your Downloads folder
2. **Right-click** on the ZIP file
3. Click **"Extract Here"**
4. You'll see a new folder appear

**Important:** Remember where this extracted folder is located! You'll need it in the next step.

## üåê Step 3: Open Chrome Extensions Page

1. Open **Google Chrome** browser
2. In the address bar (where you type websites), type exactly:
   ```
   chrome://extensions/
   ```
3. Press **Enter**

You should see a page with your installed extensions.

## üîß Step 4: Enable Developer Mode

1. Look at the **top right corner** of the extensions page
2. You'll see a toggle switch that says **"Developer mode"**
3. Click the toggle to turn it **ON** (it should turn blue or highlighted)
4. New buttons will appear - that's normal!

## üì¶ Step 5: Load the Extension

1. After enabling Developer mode, you'll see a button that says **"Load unpacked"**
2. Click the **"Load unpacked"** button
3. A file picker window will open
4. Navigate to the **extracted folder** from Step 2
5. **Select the folder** (the one that contains files like `manifest.json`)
6. Click **"Select Folder"** or **"Open"**

## ‚úÖ Step 6: Verify It Worked

1. You should see **"Agent Chat Powered by GoDaddy ANS"** appear in your extensions list
2. Make sure the toggle switch next to it is **ON** (blue/highlighted)
3. Look for the extension icon in Chrome's toolbar (top right, near the address bar)

## üéâ Done!

The extension is now installed and ready to use!

## ‚ùì Troubleshooting

### "I can't find the 'Load unpacked' button"
- Make sure Developer mode is turned ON (Step 4)
- Refresh the page and try again

### "The extension shows as 'Off'"
- Click the toggle switch next to the extension to turn it ON

### "I can't find the extracted folder"
- Check your Downloads folder
- Check your Desktop
- Try extracting the ZIP file again

### "I get an error when loading"
- Make sure you selected the **folder** (not the ZIP file)
- The folder should contain a file called `manifest.json`
- Try extracting the ZIP file again

### Still having problems?
Contact your IT support team or administrator for help.
