# GoDaddy ANS Browser Extension Wiki

Welcome to the GoDaddy ANS Browser Extension documentation!

## Overview

This Chrome extension provides an AI-powered sidebar chat with browser automation capabilities. It leverages Anthropic's Claude AI to interact with web pages through natural language commands.

## Key Features

- **AI-Powered Sidebar Chat**: Chat interface integrated into Chrome's sidebar
- **Browser Automation**: Navigate, click, type, scroll, and interact with web pages
- **Multiple AI Providers**: Support for Anthropic Claude, OpenAI, and Google Gemini
- **MCP Integration**: Model Context Protocol support for custom tools
- **Multi-Environment Builds**: Separate builds for Dev, Test, and Production environments
